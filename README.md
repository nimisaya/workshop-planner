## Tech stack

- Ruby on Rails
- CSS

## Features

Logged in users:
- Create a public or private workshop
- Clone a template workshop or a public workshop
- Logout

Logged out users:
- View public workshops and templates
- View public tasks and templates
- Create an account
- Log in
